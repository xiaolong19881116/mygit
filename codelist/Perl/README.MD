**How to install from source**

     wget http://www.cpan.org/src/5.0/perl-5.24.0.tar.gz
     tar -xzf perl-5.24.0.tar.gz
     cd perl-5.24.0
     ./Configure -des -Dprefix=$HOME/localperl
     make
     make test
     make install
